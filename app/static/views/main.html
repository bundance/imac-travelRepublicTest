momPaginator Example

by Mike Evans

The momPaginator turns your ordinary HTML table into an advanced datatable capable of retrieving data from an external ReST-based API, paginating the data, moving through the pages of data, and even sorting the data on individual columns.

This example shows a simple query of all GitHub users using the GitHub ReST API. This is a simple example, showing just the paginator and little else. For a shiner version, check out the same example with the angular-mom-spinner directive added.

Note: Annoyingly, GitHub restricts anonymous API calls to 5 calls per minute, so if you see an empty table, just wait one minute and then try again.
